API Usage Guidance

Introduction

Welcome to the API documentation for our service. This guide will help you understand how to interact with the API endpoints.

Available Endpoints

1. Get All Locations

                GET /api/locations

Retrieve all available locations.

2. Get Location Details by ID

                POST /api/location/getLocationDetailsById
                Headers:
                Content-Type: application/json

                Body:
                {
                    "ID": "Location ID"
                }

                Response:
                {
                    "message": "Location details fetched successfully",
                    "totalLocations": 1,
                    "data": [
                        {
                            "id": 1,
                            "userid": 123,
                            "name": "User Name",
                            "email": "user@example.com",
                            "status": "Active",
                            "locationAddress": "123 Main Street",
                            "latitude": "37.7749",
                            "longitude": "-122.4194",
                            "mediaPath": "http://example.com/media/path"
                        }
                    ]
                }

Retrieve detailed information for a specific location by its ID.

3. Get Locations by User ID

                POST /api/location/getLocationByUserId
                Headers:
                Content-Type: application/json

                Body:
                {
                    "userID": "User ID"
                }

                Response:
                {
                    "message": "Locations fetched successfully",
                    "totalLocations": 2,
                    "data": [
                        {
                            "id": 1,
                            "userid": 123,
                            "status": "Active",
                            "locationAddress": "123 Main Street",
                            "latitude": "37.7749",
                            "longitude": "-122.4194",
                            "mediaPath": "http://example.com/media/path"
                        },
                        {
                            "id": 2,
                            "userid": 123,
                            "status": "Inactive",
                            "locationAddress": "456 Elm Street",
                            "latitude": "40.7128",
                            "longitude": "-74.0060",
                            "mediaPath": "http://example.com/media/path2"
                        }
                    ]
                }

Retrieve all non-deleted locations associated with a specific user ID.
